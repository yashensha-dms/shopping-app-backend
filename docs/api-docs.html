Shopping App Backend API (1.0.0)

Download OpenAPI specification:

REST API for the MStore Shopping App Backend built with Laravel + Sanctum.

Authentication: Most write operations and user-specific data require a Bearer token obtained via /api/login or /api/register.

Base URL: https://<your-domain>/api
